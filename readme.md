# Daniola App

A modern commodity trading platform built with React, TypeScript, and Supabase.

## Features

- Real-time commodity trading
- Secure authentication
- Deal management
- Chat functionality
- Responsive design

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui
- Supabase
- Zustand
- React Query
- React Router
